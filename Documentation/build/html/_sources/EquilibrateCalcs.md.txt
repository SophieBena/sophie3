# Classes that perform the equilibrium calculations  

## Generic equilibrium calculator  
- [Equilibrate](EquilibrateDoc.html)  

## Configurations of [Equilibrate](EquilibrateDoc.html)    
- [EquilibrateUsingStixrude](EquilibrateUsingStixrude.html) - Uses the Stixrude-Lithgow-Bertelloni models  
- [EquilibrateUsingMELTSv102](EquilibrateUsingMELTSv102.html) - Uses the MELTS v1.0.2 model  
- [EquilibrateUsingMELTSv110](EquilibrateUsingMELTSv110.html) - Uses the MELTS v1.1.0 model  
- [EquilibrateUsingMELTSv120](EquilibrateUsingMELTSv120.html) - Uses the MELTS v1.2.0 model   
- [EquilibrateUsingpMELTSv561](EquilibrateUsingpMELTSv561.html) - Uses the MELTS v5.6.1 model 
- [EquilibrateUsingMELTSwithDEW](EquilibrateUsingMELTSwithDEW.html) - Uses the MELTS with Dew model 


## Classes that store results of the Equilibrate calculation  
- [EquilibrateState](EquilibrateState.html) (returned by [Equilibrate](EquilibrateDoc.html))  
- [EquilibrateStatePhase](EquilibrateStatePhase.html)
